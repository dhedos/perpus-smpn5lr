# **App Name**: Pustaka Nusantara

## Core Features:

- Comprehensive Data Management: Manage complete details for books (including auto-generated codes, ISBN, stock, cover images, and descriptions) and library members (students/teachers, unique ID, contact information).
- Barcode-Enabled Transaction System: Facilitate efficient book borrowing and returning processes using integrated camera-based barcode scanning, with automatic calculation of overdue status and fines.
- Offline-first with Online Synchronization: Enable full application functionality and data access even without an internet connection, with seamless automatic synchronization of all data to a central online server when connectivity is restored.
- Multi-user Authentication & Access Control: Provide a secure login system with distinct roles (Admin, Officer, Teacher, Student) and granular permissions, ensuring each user type has appropriate access to system functionalities.
- AI Book Description Tool: Utilize a generative AI tool to assist library administrators in quickly crafting or enhancing detailed book descriptions based on titles, authors, or ISBNs.
- Dashboard Analytics & Reporting: Present an intuitive dashboard displaying key library statistics such as total books, current borrowings, and popular titles, along with basic printable reports for transactions and inventory.
- Barcode & Member Card Generation: Generate and print unique barcode labels for library books for easy inventory and transaction processing, and produce printable member cards with integrated barcodes.

## Style Guidelines:

- Primary color: A sophisticated and calming blue (#2E6ECE) to reflect knowledge and reliability, offering strong contrast against the background.
- Background color: A very light, desaturated blue (#ECF0F7) providing a clean and open canvas, ideal for a professional and serene learning environment.
- Accent color: A vibrant turquoise (#33CCF7) that provides a refreshing counterpoint, drawing attention to interactive elements and calls to action without being overwhelming.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for conveying information clearly across various user roles and device types.
- Employ a consistent set of minimalist and clearly identifiable line icons to maintain a clean aesthetic and ensure quick user comprehension of functionalities.
- Implement a responsive and adaptive layout that ensures optimal usability and visual integrity across a wide range of devices, from mobile phones to desktop computers.
- Integrate subtle and smooth transition animations for navigation and data loading, contributing to a fluid user experience without sacrificing performance or speed.
